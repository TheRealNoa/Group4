**To setup:**
open cmd in the project directory
venv\\Scripts\\activate
python app.py
visit  http://localhost:5000 in your browser

If for some reason that doesn't work, try:
open cmd in the project directory
python -m venv venv
venv\\Scripts\\activate
pip install flask beautifulsoup4 requests
python app.py
visit  http://localhost:5000 in your browser
